# פונטי המותג — kurkoos-group

הפונטים בתשלום ואינם כלולים במאגר. שים כאן את הקבצים בשמות הבאים
(מומלץ woff2; אפשר להוסיף woff לתאימות), וההגדרות ב-`src/styles/fonts.css`
יטענו אותם אוטומטית.

| תפקיד | font-family | קבצים נדרשים |
|------|-------------|--------------|
| גוף — Almoni Light 300 | `Almoni` | `almoni-300.woff2`, `almoni-300.woff` |
| גוף — Almoni Regular 400 | `Almoni` | `almoni-400.woff2`, `almoni-400.woff` |
| גוף — Almoni Bold 700 | `Almoni` | `almoni-700.woff2`, `almoni-700.woff` |
| כותרות — Fb Metropolis Regular | `Fb Metropolis` | `fb-metropolis-400.woff2`, `fb-metropolis-400.woff` |
| כותרות — Fb Metropolis Bold | `Fb Metropolis` | `fb-metropolis-700.woff2`, `fb-metropolis-700.woff` |
| דקורטיבי — PowerStation Solid | `PowerStation Solid` | `powerstation-solid-400.woff2`, `powerstation-solid-400.woff` |

## מקורות רכישה
- **Almoni** — AlefAlefAlef
- **Fb Metropolis (מטרופוליס)** — fontbit / נדב עזרא: https://fontbit.co.il/product/מטרופוליס/
- **PowerStation Solid** — בתשלום

## המרה ל-woff2
אם קיבלת קבצי OTF/TTF, המר אותם (למשל בכלי אונליין "transfonter" או `fonttools`)
ושמור בשמות שבטבלה. עד שהקבצים קיימים — האתר משתמש ב-Rubik/Assistant כ-fallback.
